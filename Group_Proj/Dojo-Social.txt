# Dojo-Social
> Coding Dojo Group Project 01/2023

## Description
- User Login/Registration
- User can create/edit/delete posts
- User can create/edit/delete comments on posts
### Extras
- Socket.io Chat Feature
- AWS S3 Buckets Image Handeling
- Edit User
- Replies on comments
